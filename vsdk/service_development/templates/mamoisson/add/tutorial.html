Tutorial

To add a new language you need to add the name and an url of the picture of the flag of the country into the form. Please add the name, the audio file url, the image url of the language.

Warning, it is important that the audio files are in WAVE format. The WAVE file must have a sample rate of 8KHZ, 16 bit, mono, codec: PCM 16 LE (s16I). Otherwise the voice application cannot play the audio file.
